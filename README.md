# 🌐 Operações e Strings com JavaScript

## 📘 Sobre o Projeto
Projeto desenvolvido durante meus estudos em JavaScript com foco em entrada de dados, operações matemáticas e manipulação de strings.

## 🧠 Conceitos aplicados

- window.prompt()
- Conversão de dados com Number()
- Template strings
- Operações matemáticas
- Manipulação de strings
- length (tamanho da string)
- toUpperCase()
- toLowerCase()
- document.write()

## 💻 Funcionalidade

O projeto possui duas funcionalidades principais:

### 1️⃣ Soma de números
- Solicita dois números ao usuário
- Realiza a soma
- Exibe o resultado com alert

### 2️⃣ Manipulação de nome
- Solicita o nome do usuário
- Exibe:
  - Quantidade de letras
  - Nome em maiúsculas
  - Nome em minúsculas

## ▶️ Exemplo de execução

Digite um numero: 10  
Digite outro numero: 5  

Resultado:  
A soma entre 10 e 5 é igual a 15  

---

Qual é seu nome: Arthur  

Resultado:  
Olá, Arthur! Seu nome tem 6 letras.  
Seu nome em maiúsculas é ARTHUR  
Seu nome em minúsculas é arthur
